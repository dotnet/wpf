
!!! IMPORTANT !!!

The Generated subdirectory contains source that is generated by a tool.  Any changes to this source will be lost the next time the tool is run.  The script to generate the source can be found at %SDXROOT%\Windows\Wcp\Mil\CodeGen\GenerateFiles.cmd.

Documentation for the tool is located at http://wiki/default.aspx/Microsoft.Projects.Avalon.MilCodeGen.
